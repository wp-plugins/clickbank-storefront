== Description ==

=  Clickbank Storefront (Mall) WordPress Plugin  =

Clickbank is the World's Largest Digital Info Product Store. As a Clickbank affiliate, one can make CASH by promoting these products of any vendor who is published with Clickbank.


Using this plugin, you can host these clickbank products on your website as a wordpress plugin with just few clicks.

== Installation ==

1. Upload `clickbank-niche-storefronts-1.2.9.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Place the following **shortcode** to your store main page.

**[clickbank-storefront-bestselling]**
**[clickbank-storefront-featured]**
**[clickbank-storefront-popular]**

Now, create an another page for displaying outputs of each category of products and place the following shortcode on to it.

** [clickbank-storefront-products]**

You may visit widget section of your theme to select the category bar of your choice.Please provide 3 types of CB category bar widgets to choose from.
